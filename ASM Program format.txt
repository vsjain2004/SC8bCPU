File Extension: .asm

Program Format:

1. Instruction Memory (IMEM) section
   - Contains assembly instructions
   - One instruction per line
   - Program must end with "END"
   - Labels are supported for control flow and memory references
2. Exactly one empty line between IMEM and DMEM
3. Data Memory (DMEM) Section
   - Contains labeled data entries
   - Each line defines a variable or memory block

Data types, Number formats, and Register naming are provided in the Green Card.

For any memory address for non-branch instructions, it is recommended to use Labels and repeat them in the data memory section.
For any memory address for jump/branch instructions, it is recommended to use Labels and use them at said instruction.
Labels must be unique

Comment Lines start with // (No inline comments allowed)

For an example, Check Test.asm, and associated Test_IMEM.hex and Test_DMEM.hex
